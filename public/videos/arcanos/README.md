# Vídeos Arcano Vivo

Coloque aqui os arquivos `.mp4` verticais (9:16) de cada arcano, com o nome
correspondente ao mapeamento em `src/config/arcano-videos.ts`.

Exemplo:
- arcano-0-louco.mp4
- arcano-1-mago.mp4
- ...
- arcano-21-mundo.mp4

Se um arquivo não existir, a tela do arcano cai automaticamente para a
imagem estática da carta (fallback transparente para o aluno).
