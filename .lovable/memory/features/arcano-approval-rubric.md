# Régua de Aprovação Didática (obrigatória)

Toda revisão de arcano DEVE responder exatamente estas 5 perguntas, no formato fixo abaixo. Se qualquer resposta for NÃO, o arcano está reprovado.

## Formato obrigatório de resposta

```
ARCANO ANALISADO: [nome]

A CARTA ESTÁ CANÔNICA: SIM ou NÃO
A CARTA ESTÁ LEGÍVEL: SIM ou NÃO
A TELA ESTÁ PEDAGÓGICA: SIM ou NÃO
O TEXTO ESTÁ LENTO O BASTANTE PARA LEITURA: SIM ou NÃO
O MOBILE ESTÁ LIMPO E HIERÁRQUICO: SIM ou NÃO

ARCANO APROVADO: SIM ou NÃO
```

Se reprovado, adicionar:
```
MOTIVOS DA REPROVAÇÃO:
- [...]

AJUSTES OBRIGATÓRIOS:
- [...]
```

## Critérios de cada pergunta

**Canônica:** fidelidade RWS, composição, símbolos, enquadramento, atmosfera, ausência de releitura por IA.

**Legível:** contraste, nitidez, tamanho da carta, clareza dos textos ao redor, ausência de sobreposição.

**Pedagógica:** protagonismo da carta, hierarquia visual, clareza de título/subtítulo/blocos de apoio, experiência de estudo acima de estética.

**Texto lento o bastante:** tempo de permanência dos blocos, velocidade de animação, velocidade de digitação, tempo real de absorção desktop+mobile.

**Mobile limpo e hierárquico:** respiro, escala tipográfica, leitura sem esforço, ausência de apertos, carta como elemento principal.

## Regra de uso

- Aplicar SEM EXCEÇÃO em toda revisão arcano por arcano.
- Não aprovar atalhos. Reprovação é sinal pedagógico, não falha.
- Reusar com `mem://features/deck-fidelity-standard` (padrão técnico do deck).
