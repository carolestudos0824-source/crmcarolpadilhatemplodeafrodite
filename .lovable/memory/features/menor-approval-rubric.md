# Régua oficial — aprovação de lição de Arcano Menor

Toda lição de Menor (numerada ou corte) precisa passar nos 6 critérios para ser
considerada aprovada. Sem aprovação plena, **não escalar para a próxima carta**.

| # | Critério                  | O que verificar                                                                                   |
| - | ------------------------- | ------------------------------------------------------------------------------------------------- |
| 1 | **Fidelidade simbólica**  | Símbolos descritos batem com o RWS oficial (objetos, postura, paisagem, cores, número).           |
| 2 | **Coerência RWS**         | Leitura arquetípica + psicológica + esotérica seguem a tradição Waite-Smith, sem invenção livre.  |
| 3 | **Clareza pedagógica**    | Aluno consegue entender a carta sem precisar consultar fora; cada fase tem 1 ideia central.       |
| 4 | **Legibilidade**          | Frases curtas, hierarquia clara, sem jargão sem explicação, contraste de tipografia respeitado.   |
| 5 | **Mobile**                | Funciona em 360px sem cortes, scroll natural, carta não vaza, botões alcançáveis com o polegar.   |
| 6 | **Consistência editorial**| Tom, voz e ritmo iguais aos Maiores e às outras Menores já aprovadas (contrato dos 18 campos).    |

## Veredito padrão

Ao auditar uma carta, responder exatamente:

```
FIDELIDADE SIMBÓLICA: SIM/NÃO
COERÊNCIA RWS: SIM/NÃO
CLAREZA PEDAGÓGICA: SIM/NÃO
LEGIBILIDADE: SIM/NÃO
MOBILE: SIM/NÃO
CONSISTÊNCIA EDITORIAL: SIM/NÃO
LIÇÃO APROVADA: SIM/NÃO
```

Só com **6/6 SIM** a carta entra no baseline e libera a próxima.
