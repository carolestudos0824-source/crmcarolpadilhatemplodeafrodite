# Project Memory

## Core
Platform name: A Jornada do Louco. Tagline: Onde o tarô se revela. Promise: O tarô tem profundidade. Seu estudo também deveria ter.
Light parchment theme: ivory bg, gold primary, crimson secondary. Cinzel headings, Inter body, Cormorant Garamond accent.
Tarot learning platform with gamification (Duolingo-inspired, not childish). Premium, symbolic, inclusive.
Lessons use 5-layer pedagogy: main (required) → deepDive → extras → exercise → quiz. Students advance without consuming optional content.
Base simbólica: Rider-Waite-Smith. Leituras arquetípicas, psicológicas e esotéricas. Coerência em todas as lições.
Arcanos use ArcanoMaiorEditorial type (17 fields). Files in src/data/arcanos/. 22/22 complete.
Copas is the first complete Minor Arcana suit — 14/14 cards filled. Files: copas-1-5.ts, copas-6-10.ts, copas-corte.ts.
Linguagem inclusiva: nunca limitar público a "mulheres". Usar neutro: estudantes, boas-vindas, quem quer aprender.
